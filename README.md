# Mini-Paint-Project (Java OOP Project)

## Описание
Mini-Paint — это кроссплатформенное десктопное приложение для рисования. 
Оно начинается с консольного интерфейса, где пользователь настраивает параметры инструмента (размер и цвет), после чего открывается графический холст для свободного рисования с помощью мыши.

## Используемые технологии
- Язык: Java (JDK 11+)
- Интерфейс: Java Swing / AWT
- Парадигма: Строгое Объектно-Ориентированное Программирование (ООП)

## Реализованные принципы ООП
1. Инкапсуляция (private поля, геттеры/сеттеры).
2. Наследование (Класс Brush наследует Tool).
3. Полиморфизм (Переопределение @Override метода apply()).
4. Абстракция (Абстрактный класс Tool с абстрактным методом).
5. Интерфейсы (MouseMotionListener).
6. Обработка исключений (try-catch для защиты консольного ввода).
